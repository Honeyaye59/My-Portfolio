UNO Pay

အသုံးပြုသူ တစ်ယောက်နဲ့တစ်ယောက်ကြား ပေးစရာရှိတာတွေ ရစရာရှိတာတွေကို မှတ်ပေးတဲ့ project တစ်ခုဖြစ်ပါတယ်။

  • အုပ်စုဖွဲ့ card game တွေဆော့တဲ့အခါမျိုးမှာ တစ်ယောက်နဲ့တစ်ယောက် ရစရာရှိတာ ပေးစရာရှိတာကို စာရွတ်နဲ့မှတ်တာဖြစ်ဖြစ် ဦးနှောက်နဲ့မှတ်တာဖြစ်ဖြစ် လုပ်စရာမလိုပဲ ဒီ appက တစ်ခါထဲ money to give, money to receive ဆိုပြီး အသုံးပြုသူတွေက တစ်ယောက်နဲ့တစ်ယောက် money transfer လုပ်လိုက်တာနဲ့ တစ်ခါတည်း တွက်ချက်ပေးသွားမှာ ဖြစ်ပါတယ်။
  • Card gameတွေဆော့တဲ့အခါ ပိုပြီးfun ဖြစ်စေဖို့ ပေးရမယ့်ငွေ ရစရာရှိတာတွေကို မှတ်ရသက်သာစေဖို့ ရည်ရွယ်ပါတယ်။
  • Card gameတွေမှာမှမဟုတ်ပဲ သူငယ်ချင်းတွေအပြင်သွားတဲ့အခါ ပိုက်ဆံပေါင်းရှင်းပြီး ပြီးမှပြန်တွက်ချင်တဲ့အခါမျိုးအတွက်လည်း အသုံးပြုနိုင်ပါတယ်။

User Guide

  • Responsive ဖြစ်တဲ့အတွက် mobile မှာရော desktop မှာရော ကြိုက်တဲ့ deviceနဲ့အသုံးပြုနိုင်ပါတယ်။

  • အရင်ဆုံး user အသစ်ဆို register လုပ်ရပါမယ်။ Username က unique ဖြစ်ရပါမယ် Displayname ကတော့ ကြိုက်ရာပေးလို့ရပါတယ် email ကလည်း unique ဖြစ်ရပါမယ်။ Password က character 8 လုံးရှိရပါမယ်။ အားလုံးကို ဖြည့်ပြီးမှ register လုပ်လို့ရပါမယ်။

  • Register တစ်ခါလုပ်ဖူးပြီးရင်တော့ login လုပ်ရင် ရပါတယ် login လုပ်ဖို့အတွက် register လုပ်ထားတဲ့ email နဲ့ password လိုအပ်ပါတယ် နှစ်ခုလုံး ကိုက်ညီမှုရှိမှ app ထဲဝင်လို့ရမှာပဲဖြစ်ပါတယ်

  • ဒါကတော့ User ရဲ့ Home Page ပဲဖြစ်ပါတယ် User ရဲ့ walletထဲမှာရှိတဲ့ amount ကိုဖော်ပြထားပါတယ် အစကပြန်စချင်ရင် refresh wallet ဆိုပြီး 0 ကနေပြန်စလို့ရပါတယ် currency ကိုလည်းရွေးလို့ရပါတယ် ဒီထဲမှာတော့ Dollar နဲ့ Myanmar Kyat ကိုထည့်ထားပါတယ်

    Transfer ဆိုတဲ့ box လေးကိုတော့ တခြားအသုံးပြုသူတွေကို money transfer လုပ်ချင်တဲ့ အခါအသုံးပြုပါတယ် Groups ကိုတော့ ကိုယ့်ရဲ့ group တွေကိုကြည့်ချင်ရင် နှိပ်လို့ရပါတယ် Transfer History ကိုတော့ ကိုယ် transfer လုပ်ခဲ့တဲ့ history တွေကို ပြန်ကြည့်နိုင်ပါတယ်

  • Input Box မှာ ကိုယ် ပို့ချင်တဲ့သူရဲ့ username အစစာလုံးကို ရေးလိုက်တာနဲ့ သက်ဆိုင်ရာ user တွေရဲ့နာမည်တွေကျလာပါလိမ့်မယ်

    ကိုယ်ပို့ချင်တဲ့သူရဲ့ နာမည်ကို နှိပ်ပြီး ပိုက်ဆံပမာဏ ကိုဖြည့်ပြီးရင် ပို့လို့ရပါပြီ တကယ်လို့ ဘာမှမဖြည့်ထားရင်တော့ transfer button ကအလုပ်လုပ်မှာမဟုတ်ပါ

  • Transfer History တွေကို dateအလိုက်ရွေးပြီး filterလုပ်လို့ရပါတယ်

  • ဘေးက menu bar မှာတော့ user နဲ့သက်ဆိုင်တဲ့ အချက်အလက်ရယ် app setting နဲ့ account setting ဆိုပြီး ရှိပါတယ်

  • Groups ကိုနှိပ်လိုက်ရင် ကိုယ့်ရဲ့ သက်ဆိုင်ရာ groupတွေကိုတွေ့နိုင်ပါတယ် တစ်ခုချင်းစီနှိပ်ပြီး group detail ကိုဝင်ကြည့်လို့ရပါတယ် edit နှိပ်ပြီး group name ပြောင်းလို့ရသလို members တွေကိုလည်း removeလုပ်လို့ရပါတယ်

  • Group Detail မှာဆို groupထဲမှာရှိတဲ့ တခြားmemeber တွေနဲ့ ဆက်သွယ်မှုဖြစ်တဲ့ ပေးရမယ့်ငွေ ရရမယ့်ငွေတွေကို ဖော်ပြထားပါတယ် groupထဲက member တွေကို transfer လုပ်ထားတဲ့ history member တွေကနေ transfer လုပ်ထားတဲ့ historyတွေကိုလည်း ဖော်ပြထားပါတယ်

  • App Setting မှာဆိုရင် Light Mode ကနေ Dark Modeပြောင်းလို့ရပါတယ် Language ကိုလည်း မြန်မာဘာသာမှ အင်္ဂလိပ်ဘာသာ ပြောင်းနိုင်ပါတယ်

  • Account Setting မှာဆိုရင် Profile Picture, Username, Displayname ပြောင်းနိုင်ပြီး Log outလည်းလုပ်နိုင်ပါတယ်

Developer Guide

  • Frontend framework မှာတော့ Angular ကိုအသုံးပြုထားပြီး Responsive Design အတွက် Bootstrap ကို အသုံးပြုထားပါတယ်။ UI အတွက်တော့ figma နဲ့ဆွဲထားပါတယ်။ backend အတွက် ERD diagram ဖြစ်ပါတယ်။ဆက်ပြီးတော့ features တွေထပ်ထည့်မှာဖြစ်ပြီး Real world project အဖြစ် upgrade လုပ်သွားမှာပဲဖြစ်ပါတယ်။
