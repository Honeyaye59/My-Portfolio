တွဲဖက်မစားသင့်သော အစားအစာများကို ဖော်ပြပေးသော website ဖြစ်ပါတယ်။

  • ဘယ်ဘက်မှာ အားလုံးကိုဖော်ပြထားပြီး ညာဘက်မှာတော့ filter လုပ်ပြီး ကြည့်နိုင်အောင် ရောဂါလက္ခဏာ အလိုက် linkလေးတွေရှိပါတယ်။ ညာဘက်က link တွေကိုနှိပ်ပြီး သက်ဆိုင်ရာ ရောဂါလက္ခဏာတွေကို filter လုပ်ပြီးကြည့်နိုင်ပါတယ်။

  • Pagination button တွေကို pure CSS ရယ် vanilla JavaScript နဲ့ပဲရေးထားပါတယ်။

    items တွေကို page အလိုက်ခွဲထားပါတယ်

    အရင်ဆုံး website ကိုဝင်လိုက်ရင် page one ကနေစပြီး page တစ်ခုစီမှာ items 10 ခုစီ ပြထားပါတယ် item ရှိသလောက် page တွေခွဲထားခြင်းဖြစ်ပါတယ်။

    user page 1 ရောက်နေရင်တော့ previous button ကို disabled (နှိပ်လို့မရအောင်) လုပ်ထားပါတယ် user ကနောက်ဆုံး page ကိုရောက်နေရင် next button ကို disabled (နှိပ်လို့မရအောင်) လုပ်ထားပါတယ်

  • ညာဘက်က မူးဝေ ဆိုတဲ့ link ကိုနှိပ်လိုက်ရင် ဘယ်ဘက်မှာ မူးဝေ ရောဂါလက္ခဏာ ရှိတဲ့ items တွေပဲကျလာပါလိမ့်မယ်

  • Pagination button တွေကို page ရှိသလောက်ပဲ ဖော်ပြပေးထားပါတယ် page 2 ခုရှိရင် button 1 နဲ့ 2 နဲ့ ပဲရှိပါမယ်

  • Responsive ဖြစ်လို့ မည်သည့် device နဲ့မဆိုအသုံးပြုနိုင်ပါတယ်
